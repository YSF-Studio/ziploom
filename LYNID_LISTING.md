## 📄 Full Description

### What is ZipLoom?

ZipLoom is a modern, privacy-first archive utility built entirely in Rust — no external tools, no dependencies, no network calls. Everything runs locally on your machine.

Perfect for: developers, sysadmins, security researchers, and anyone who needs reliable archive operations with a side of forensic analysis.

### ✨ Features

**📦 Archive Operations**
- **Compress:** ZIP, TAR, TAR.GZ, TAR.BZ2, TAR.XZ, TAR.ZST
- **Extract:** ZIP, TAR, TAR.GZ, TAR.BZ2, TAR.XZ, TAR.ZST, 7z, RAR
- **AES-256 encryption** for password-protected ZIP files
- **Compression levels 0–9** — fine-tune speed vs ratio
- **Split archives** — compress and split into volume chunks
- **Clean macOS junk** — auto-strips .DS_Store, __MACOSX, ._ files

**🔍 Forensic Inspector**
- **Magic byte verification** — detects tampered files and format mismatches
- **Entropy analysis** — spots encrypted or compressed content inside archives
- **Batch file hashing** — MD5, SHA-1, SHA-256 per entry
- **Anomaly detection** — flags suspicious files automatically
- **Export CSV forensic reports**

**🛡️ Privacy & Security**
- ✅ 100% offline — zero network calls
- ✅ No telemetry, no data collection
- ✅ Pure Rust backend — memory-safe, no shell commands
- ✅ No external dependencies (no 7z, bsdtar, or unrar needed)
