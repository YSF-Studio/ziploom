# Screenshot Guide for ZipLoom Marketing

> Take these screenshots from the running app and use them for Lynkid listing, GitHub README, and social media.

## 📸 Screenshot 1 — Compress Tab (Hero Image)
**Use for:** Lynkid listing cover, GitHub README hero

**What to show:**
1. Open ZipLoom
2. Tab "Compress" active (first tab)
3. Add 3–4 sample files (mix of images, documents, and a folder)
4. Select format "ZIP"
5. Enable password (show AES-256 notice)
6. Set compression level to "6 / Balanced"
7. **Capture** — shows drag-drop area, file list, format dropdown, password toggle, compression slider

*Filename: `screenshot-compress.png`*

## 📸 Screenshot 3 — Forensic Inspector (Main View)
**Use for:** Lynkid gallery, Reddit/HN posts

**What to show:**
1. Click "Inspect" tab
2. Open a complex archive (ZIP with mixed files: .txt, .jpg, .pdf, .docx)
3. Enable columns: Name ✅, Size ✅, Entropy ✅, Magic ✅, Hashes ❌
4. Sort by name
5. **Capture** — shows file tree with colored entropy bars, magic byte status (✅/⚠️/❌)

*Filename: `screenshot-forensic-main.png`*
